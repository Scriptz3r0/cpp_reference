## Algorithm

### code-example

```cpp
#include <algorithm>
#include <iostream>

std::vector<string> names = {"AAA", "BBB" , "CCC"};
sort(names.begin(), names.end());

```

- Doesn't work with lists.
- Works with, for example, vectors.
